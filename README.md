# wordpress-to-firebase
import data from wordpress to firebase

### install dependencies
```
$ npm install
or
$ yarn
```

### dotenv variables
create file ```.env``` and set the variables with your firebase key:
```
DOMAIN = yourdomain.com
apiKey =
authDomain =
databaseURL =
storageBucket =
messagingSenderId =
```

### run
```
node index.js
```
